## My Hero Academia - Fandom

This project is a fandom page for the anime My Hero Academia. It uses the canvas element to create a eraser element that erases the background to reveal the video. The site is fully responsive and uses framer motion for animations. The site uses the intersection observer to tell you when an element enters or leaves the viewport and then lazy loads the video. The site is built using Next.js and is deployed on Netlify.

[Visit Website](https://moni-mha.netlify.app/)

### Stack

- [Next.js](https://nextjs.org/) - Next.js enables you to create full-stack Web applications by extending the latest React features.
- [Framer Motion](https://www.framer.com/motion/) - Framer Motion is a production-ready motion library for React, React Native, and Vue.
- [Styled Components](https://styled-components.com/) - Styled Components is a library for React that allows you to write actual CSS code to style your components.
- [React Intersection Obeserver](https://www.npmjs.com/package/react-intersection-observer) - It is a React component that can be used to observe when it enters or leaves the viewport.
