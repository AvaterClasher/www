## Ball Race - A Interactive 3D Experience

This project leverages React Three Fiber for 3D game development, integrating the Rapier physics engine for realistic interactions. Zustand is employed for state management, while r3f-perf monitors performance metrics. Postprocessing is applied for visual effects, drei facilitates the creation of a skybox, and Vite.js ensures a rapid development environment. The final deployment is hosted on Vercel.

[Visit Website](https://moni-game.vercel.app/)

### Stack

- [Vite.js](https://vitejs.dev/) - Vite is a build tool that aims to provide a faster and leaner development experience for modern web projects.
- [@react-three/fiber](https://github.com/pmndrs/react-three-fiber) - React Three Fiber is a React renderer for Three.js.
- [@react-three/drei](https://github.com/pmndrs/drei) - React Three Fiber's companion library.
- [@react-three/postprocessing](https://github.com/pmndrs/react-postprocessing) - React Three Fiber's postprocessing library.
- [@react-three/rapier](https://github.com/pmndrs/react-three-rapier) - React Three Fiber's physics library.
- [r3f-perf](https://github.com/pmndrs/r3f-perf) - A performance profiler for React Three Fiber.
- [zustand](https://github.com/pmndrs/zustand) - Zustand is a small, fast and scalable bearbones state-management solution.
