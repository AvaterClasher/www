## Jaif

Jaif is a utility program designed to convert PNG images to a custom JAIF format and vice versa. The JAIF format stores pixel data in hexadecimal RGB format, making it a specialized format for certain applications. In addition, Jaif includes functionality to display images using the Fyne GUI framework.

[Visit Repository](https://github.com/AvaterClasher/jaif)

### Features

- **Image Conversion**: Convert PNG images to JAIF format and back.
- **Hexadecimal RGB Storage**: JAIF format stores pixel data in hexadecimal RGB format.
- **Image Display**: Use the Fyne GUI framework to display converted images.
- **Go-Based**: Written in Go, leveraging popular libraries for image processing and GUI.
