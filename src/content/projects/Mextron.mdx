## Mextron

Mextron is a static documentation generator written in Rust. It leverages various Rust libraries to provide efficient and flexible documentation generation. The implementation includes features such as asynchronous operations, templating, and file system utilities. The deployment is hosted on Netlify.

[Visit Website](https://mextron.vercel.app)

### Stack

- [axum](https://docs.rs/axum) - Web framework that focuses on ergonomics and modularity.
- [chrono](https://docs.rs/chrono) - Date and time library.
- [clap](https://docs.rs/clap) - Command-line argument parsing.
- [handlebars](https://docs.rs/handlebars) - Templating engine.
- [html-minifier](https://docs.rs/html-minifier) - HTML minification.
- [minify-html](https://docs.rs/minify-html) - HTML minification in Rust.
- [pulldown-cmark](https://docs.rs/pulldown-cmark) - CommonMark parser.
- [serde](https://docs.rs/serde) - Serialization and deserialization framework.
- [serde_json](https://docs.rs/serde_json) - JSON serialization and deserialization.
- [sitewriter](https://docs.rs/sitewriter) - Static site generator.
- [tokio](https://docs.rs/tokio) - Asynchronous runtime for Rust.
- [tower-http](https://docs.rs/tower-http) - Utilities for working with Tower, a library for building robust and scalable Web services.

### Features

- **Asynchronous Operations**: Efficiently handle I/O operations with async/await.
- **Templating**: Use Handlebars for flexible templating.
- **File System Utilities**: Leverage various utilities for interacting with the file system.
- **HTML Minification**: Optimize HTML files with minification.
- **Multi-format Support**: Supports JSON, YAML, and TOML formats.
