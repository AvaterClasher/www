## Gat

Gat is a Go implementation of the Unix cat command used to concatenate file(s) and print the result to the terminal. It replicates the functionality of the Unix cat command with additional features for enhanced output control.

[Visit Repository](https://github.com/AvaterClasher/gat)

### Features

- **Concatenate**: Combine multiple files and output their content to the terminal.
- **Read from Standard Input**: Use standard input if no files are provided or **-** is used.
- **Number Lines**: Optionally number all lines or only non-blank lines.
- **Suppress Repeated Empty Lines**: Remove consecutive empty lines from the output.
- **Display End-of-Line Characters**: Show end-of-line characters for better readability.
- **Show Tab Characters**: Explicitly display tab characters.

### Dependencies

- **Standard Library**: Utilizes Go's standard library packages for terminal I/O and file handling.
