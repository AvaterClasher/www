## MacOS

This web application serves as a browser-based implementation of the well-known MacOS. Developed using Next.js and Tailwind CSS, it is a static website that replicates essential operating system functions. The implementation includes functional components such as Spotify, a browser, and a terminal. Furthermore, a working calculator and notepad are integrated into the system. The deployment is hosted on Netlify.

[Visit Website](https://moni-mac.netlify.app/)

### Stack

- [Next.js](https://nextjs.org/) - Next.js enables you to create full-stack Web applications by extending the latest React features.
- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework.
- [Jquery](https://jquery.com/) - A JavaScript library that simplifies HTML DOM tree traversal and manipulation.
- [React-Draggable](https://github.com/react-grid-layout/react-draggable) - A React component for drag and drop.
- [Netlify](https://netlify.app) - Hosting platform for static websites.
