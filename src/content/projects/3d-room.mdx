## 3D Room - A Interactive 3D Experience

A interactive 3D experience that utilizes Three.js to render 3D models in a web browser, employing the GLTFLoader to load models and GSAP for animations. Tweakpane is used for interactive control of model textures and day-night environment settings. The models, with baked textures created in Blender, respond to mouse input for rotation. Webpack bundles the project files, and the final deployment is on Netlify for a seamless web experience.

[Visit Website](https://moni-3d-room.netlify.app/)

### Stack

- [Vite](https://vitejs.dev/) - Vite is a build tool that aims to provide a faster and leaner development experience for modern web projects.
- [Webpack](https://webpack.js.org/) - Webpack is a static module bundler for modern JavaScript applications.
- [Gsap](https://greensock.com/gsap/) - GSAP is a robust JavaScript animation library that allows developers to easily create high-performance animations for web and mobile applications.
- [Three.js](https://threejs.org/) - Three.js is a powerful, feature-rich, and easy-to-use JavaScript library for WebGL.
